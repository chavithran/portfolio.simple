🎨 Personal Portfolio Website

This repository contains my Portfolio Website, built using HTML and basic CSS. The goal of this project is to create a simple, clean, and visually appealing website to showcase my profile and allow others to contact me.

🔗 Pages Included Home Page – Introduces who I am, my skills, and a brief overview of my work. Contact Page – Contains contact information and a simple layout for visitors to reach out. 🚀 Technologies Used HTML5 CSS3 (Basic styling) 💡 Purpose

This project helped me improve my understanding of webpage structure, layout design, and basic styling. It serves as the foundation for building more advanced portfolio websites in the future.
